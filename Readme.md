# 💻 PICSEL - CSE Committee Website (2025–2026)

This repository contains the official website for the **PICSEL (Panel in Computer Science and Engineering)** committee for the academic session **2025–2026**.

The website serves as a central platform to showcase:
- 📅 Events conducted by the committee  
- 👥 Committee members and faculty  
- 🏫 Department highlights  
- 📢 Announcements and updates  

---

## 🚀 About PICSEL

PICSEL is the student-driven technical forum of the **Computer Science and Engineering Department**. It focuses on fostering innovation, collaboration, and technical growth among students.

- 📌 Clean and modern UI  
- 📅 Event listings and details
